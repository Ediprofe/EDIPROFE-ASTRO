# **Taller 1: Separación de mezclas**

---

## **Ejercicio 1**

Una mezcla está conformada por **dos o más componentes**.
Puede ser:

* **Heterogénea**, si sus componentes se distinguen a simple vista.
* **Homogénea**, si sus componentes no se diferencian.

A continuación, se presenta una tabla con algunos **métodos de separación** y la propiedad física que aprovechan.

---

### **Métodos de separación de mezclas**

| **Método**      | **Descripción**                                                                   |
| --------------- | --------------------------------------------------------------------------------- |
| **Tamizado**    | Permite separar componentes **sólidos de diferentes tamaños**.                    |
| **Filtración**  | Separa **mezclas heterogéneas** con componentes **sólidos y líquidos**.           |
| **Decantación** | Permite separar **líquidos** con **diferentes densidades**.                       |
| **Evaporación** | Permite separar un **sólido disuelto en un líquido** en una **mezcla homogénea**. |

---

### **Pregunta**

De acuerdo con la información anterior, si se desea separar una mezcla de **azúcar disuelto en agua**, ¿cuál es el método de separación adecuado?

A. Filtración
B. Tamizado
C. Evaporación
D. Decantación

## **Ejercicio 2**

Un estudiante quiere determinar los componentes de un tipo de tinta utilizando el **método de cromatografía en papel**, el cual se basa en la **separación de los componentes** de una sustancia.
Para esto, coloca una **gota de tinta** sobre un **papel filtro**, que a su vez se pone en contacto con un **solvente**, como se muestra en la figura.

A medida que el solvente **asciende por el papel**, comienzan a aparecer **manchas de diferentes colores**, cada una correspondiente a un componente de la tinta.

### **Inicio del experimento**

```
           ┌───────────────────────────┐
           │        Barra soporte      │
           └──────────────┬────────────┘
                          │
                          │
                          ●   ← Gota de tinta
                          │
                ┌───────────────────────┐
                │         Papel         │
                │        filtro         │
                │                       │
                ├───────────────────────┤  ← Nivel del solvente
                │                       │
                │       Solvente        │
                │                       │
                └───────────────────────┘
```

---

### **Después de la separación**

```
           ┌───────────────────────────┐
           │        Barra soporte      │
           └──────────────┬────────────┘
                          │
                          │
                ┌─────────┼─────────┐
                │   ███   │   Mancha 1 (color 1)
                │   ███   │
                │   ▒▒▒   │   Mancha 2 (color 2)
                │   ▒▒▒   │
                │   ▓▓▓   │   Mancha 3 (color 3)
                │   ▓▓▓   │
                ├─────────┼─────────┤  ← Solvente
                │                       │
                │       Solvente        │
                │                       │
                └───────────────────────┘
```
Según el resultado, ¿qué es la tinta?

A. Una mezcla heterogénea.
B. Una sustancia pura.
C. Un compuesto.
D. Una mezcla homogénea.

## Ejercicio 4

Una estudiante quiere hacer una crema hidratante y para esto realiza una búsqueda y encuentra varias formas de hacerla con materiales que tiene disponibles en su casa. Uno de los procedimientos dice que debe mezclar dos tazas de agua con una cucharada pequeña de jabón de manos y una taza de aceite mineral. La estudiante cometió un error y mezcló primero el agua con el aceite.

De los siguientes procedimientos, ¿cuál puede aplicar para separar el aceite de la mezcla?

A. Pasar la mezcla a un recipiente transparente; esperar a que se separe en dos fases y el aceite será el que esté en mayor proporción.

B. Agitar la mezcla y separarla por igual en dos recipientes; el primero tendrá el agua y el segundo el aceite.

C. Pasar la mezcla a un recipiente transparente; esperar a que se separe en dos fases y el aceite será el que esté en menor proporción.

D. Agitar la mezcla y separarla por igual en dos recipientes; el primero tendrá el aceite y el segundo el agua.
